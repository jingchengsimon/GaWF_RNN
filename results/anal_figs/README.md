# Analysis figures

Categorized figures live directly under the category tree
`results/anal_figs/<CATEGORY>/`; there are no script subdirectories below the category. Existing
unclassified legacy figures remain directly below this root until they receive an explicit
category. The `archive/` directory is reserved for figures that are intentionally excluded from
the main result set. When different runs produced the same basename, the colliding files retain
their source run tag as a prefix so that flattening never overwrites an artifact.
